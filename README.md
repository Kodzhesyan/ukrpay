
# UkrPay

**Платити легко.**

Професійний інструмент для генерації платіжних QR-кодів згідно зі стандартами Національного банку України.

## Як розгорнути на GitHub Pages:

1. Створіть новий репозиторій на GitHub.
2. Завантажте всі файли проекту.
3. Перейдіть у **Settings** -> **Pages** вашого репозиторію.
4. У розділі **Build and deployment** -> **Source** виберіть **GitHub Actions**.
5. Зробіть `git push` у гілку `main`.
6. GitHub автоматично збере проект і за кілька хвилин опублікує його.

## Особливості:
- Повна відповідність стандарту НБУ (BCD/002).
- Підтримка Deep Linking (передача даних через URL).
- Збереження реквізитів у браузері (LocalStorage).
- Генерація QR з патріотичним логотипом гривні та завантаження у PNG.
